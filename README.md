# Gemini Chat Organizer

Google Gemini Webのチャットをタグでグループ分け・フィルタリングするChrome拡張機能。

GWS（Google Workspace）版Geminiではチャット削除ができないため、増え続けるチャットの整理に特化しています。

## 機能

- **タグフィルタ** - `[タグ名]`形式のチャットタイトルを自動認識してグループ化
- **カスタムタグ** - タイトルを変えずにタグを後付け（カンマ区切りで複数タグ対応）
- **検索** - チャットタイトルのインクリメンタル検索
- **非表示（疑似削除）** - 不要なチャットをサイドバーから非表示に（復元可能）
- **ポップアップ管理画面** - ドラッグ&ドロップでタグ間移動
- **chrome.storage.sync** - タグ・非表示設定をGoogleアカウントで同期

## インストール

### 開発者モードで読み込み

1. このリポジトリをクローン
   ```
   git clone https://github.com/YOUR_USERNAME/GeminiWebSearcher.git
   ```
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンしたフォルダを選択

## 使い方

### タグの自動認識
チャットタイトルの先頭に`[タグ名]`を付けると自動でグループ化されます。
```
[Sales]クオリテム        → Salesグループ
[Engineer]AIエージェント → Engineerグループ
[Fill][PCG]VR施工管理    → FillとPCGの両方に表示
```

### カスタムタグの付与
1. サイドバーのチャットにマウスを乗せる
2. 🏷️ ボタンをクリック
3. タグを入力して保存（カンマ区切りで複数可）

### チャットの非表示
1. サイドバーのチャットにマウスを乗せる
2. × ボタンをクリック
3. 「非表示を表示」ボタンで復元可能
